Painel de Notas da Turma

Como usar:
1. Abra o arquivo index.html no navegador.
2. Clique em "Escolher planilhas".
3. Selecione uma ou várias planilhas .xlsx no modelo das turmas.
4. Use o campo "Ver turma" para alternar entre as turmas carregadas.
5. Veja o painel da turma, filtre os alunos e clique em "Relatório".
6. Use "Imprimir relatório" para imprimir apenas o relatório individual.

Modelo esperado da planilha:
- Cabeçalho com Ano/Turma, por exemplo: Ano/Turma: 6A
- Colunas: Número aluno, Av1, Av2, Média, Final, Observações
- Rodapé com Considerações sobre a turma

Observação:
Para funcionar sem internet, a pasta js precisa conter o arquivo xlsx.full.min.js.
